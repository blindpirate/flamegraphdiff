Usage instructions

flamegraphdiff can be installed by cloning the git repository and the submodules:

$ git clone https://github.com/corpaul/flamegraphdiff.git --recursive

Generating a differential flame graph is easy! Simply run:

$ cd flamegraphdiff
$ python graphs/diff_flame_graphs.py output/example_v1.txt output/example_v2.txt output/example_diff.txt
$ cat output/example_v1.txt | FlameGraph/flamegraph.pl > output/example_v1.svg
$ cat output/example_v2.txt | FlameGraph/flamegraph.pl > output/example_v2.svg
$ cat output/example_diff.txt | FlameGraph/flamegraph.pl > output/example_diff.svg
$ python graphs/generate_dfg_report.py output/example_v1.svg output/example_v2.svg output/example_diff.svg
